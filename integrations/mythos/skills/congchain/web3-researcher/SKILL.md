# Web3 Researcher

Use this skill for Web3 due diligence, Solana ecosystem research, protocol
analysis, tokenomics review, and market intelligence.

## Safety Contract

- This skill does not provide financial advice.
- Do not tell users to buy, sell, stake, bridge, borrow, lend, or farm.
- Do not connect wallets or request wallet credentials.
- Verify claims through at least two public sources when possible.
- Label time-sensitive data with the date and source.

## Primary Sources

- Project docs and official repositories.
- Solscan, Solana Explorer, SolanaFM.
- DefiLlama, Jupiter stats, Birdeye, Dexscreener.
- Security reports from known auditors when available.
- Public governance forums and official social channels.

## Workflow

1. Define the research question and time window.
2. Identify project, token, program ID, repository, and documentation.
3. Collect public metrics: TVL, volume, holders, program authority, audits,
   multisig, upgrade status, liquidity, and dependency risk.
4. Separate objective evidence from inference.
5. Produce a risk-labeled report.
6. Offer to save only the reviewed summary as CongChain memory.

## Pitfalls

- TVL can be inflated.
- Token distribution can change quickly.
- Explorer labels can be wrong or incomplete.
- Anonymous teams are not automatically malicious, but increase review burden.
- Old audits may not cover the deployed code.
